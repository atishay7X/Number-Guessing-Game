# Number-Guessing-Game
Number Guessing Game in Python(Beginner Friendly Project)


Features included:

3 difficulty levels — Easy (1–50, 10 guesses), Medium (1–100, 7 guesses), Hard (1–200, 5 guesses)
Hot/cold hints — tells you if you're a little off or way off
Input validation — handles non-numbers and out-of-range guesses without wasting a turn
Play again prompt at the end
